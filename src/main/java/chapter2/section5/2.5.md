
# 稳定性
| 算法     | 是否稳定 | 是否为原地排序 | 时间复杂度             | 空间复杂度 |
|--------|------|---------|-------------------|-------|
| 选择排序   | 否    | 是       | N^2               | 1     | 
| 插入排序   | 是    | 是       | 介于N与N^2           | 1     | 
| 希尔排序   | 否    | 是       | N*log(N) Or N^(6/5) | 1     | 
| 快速排序   | 否    | 是       | N*log(N)          | lg(N) | 
| 三项快速排序 | 否    | 是       | 介于N和N*log(N) 之间   | lg（N） | 
| 归并排序   | 是    | 否       | N*log(N)   | N     | 
| 堆排序    | 否    | 是       | N*log(N)   | 1     |
